# Book17

Book17 is a lightweight appointment booking Micro-SaaS for service-based businesses such as barbers, salons, tutors, and trainers.

## Features
- Business authentication (JWT)
- Public appointment booking page
- Availability & time-slot validation
- Booking dashboard
- 7-day free trial system
- Paystack & Flutterwave integration (test mode)
- SMS notification placeholder
- Mobile-first UI

## Tech Stack
- Backend: Node.js, Express, MongoDB
- Frontend: React
- Payments: Paystack (Test), Flutterwave (Test)
- Hosting: Render, Vercel
